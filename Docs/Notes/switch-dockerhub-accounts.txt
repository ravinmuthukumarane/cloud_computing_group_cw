Switch Docker Hub accounts
==========================

Docker Hub accounts:

1. ravinmuthukumarane
   Docker config folder: ~/.docker
   This is the normal/default Docker folder already on this server.

2. nipuniamarasinghe
   Docker config folder: ~/.docker-nipuniamarasinghe


One-time setup
--------------

Keep the existing Docker folder for ravinmuthukumarane:

~/.docker

Create a separate Docker config folder for nipuniamarasinghe:

mkdir -p ~/.docker-nipuniamarasinghe


Login as ravinmuthukumarane
---------------------------

docker login -u ravinmuthukumarane


Login as nipuniamarasinghe
--------------------------

DOCKER_CONFIG=~/.docker-nipuniamarasinghe docker login -u nipuniamarasinghe


Use ravinmuthukumarane
----------------------

docker build -t ravinmuthukumarane/IMAGE_NAME:TAG .
docker push ravinmuthukumarane/IMAGE_NAME:TAG


Use nipuniamarasinghe
---------------------

DOCKER_CONFIG=~/.docker-nipuniamarasinghe docker build -t nipuniamarasinghe/IMAGE_NAME:TAG .
DOCKER_CONFIG=~/.docker-nipuniamarasinghe docker push nipuniamarasinghe/IMAGE_NAME:TAG


Tag an existing image for ravinmuthukumarane
--------------------------------------------

docker tag LOCAL_IMAGE:TAG ravinmuthukumarane/IMAGE_NAME:TAG
docker push ravinmuthukumarane/IMAGE_NAME:TAG


Tag an existing image for nipuniamarasinghe
-------------------------------------------

docker tag LOCAL_IMAGE:TAG nipuniamarasinghe/IMAGE_NAME:TAG
DOCKER_CONFIG=~/.docker-nipuniamarasinghe docker push nipuniamarasinghe/IMAGE_NAME:TAG


Check current normal Docker config
----------------------------------

docker info


Important
---------

The Docker Hub namespace must match the account you want to push to.

For ravinmuthukumarane:

ravinmuthukumarane/image-name:tag

For nipuniamarasinghe:

nipuniamarasinghe/image-name:tag
